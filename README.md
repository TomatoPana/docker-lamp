# docker-lamp
Apache - PHP7 - Mysql5.7 - phpmyadmin - redis - portainer

## Supported Containers
- **Database:**
	- MySQL
- **No Sql:**
	- Redis
- **WEB:**
	- PHP7 Apache2
- **Tools:**
	- PhpMyAdmin
	- Portainer

change .env to add your local settings for web folders and ports to expose the services and then run:

docker-compose up -d --build
