# docker-lamp
Apache - PHP7 - Mysql5.7 - phpmyadmin - redis

## Supported Containers
- **Database:**
	- MySQL
- **No Sql:**
	- Redis
- **Servers:**
	- PHP7 Apache2
- **Tools:**
	- PhpMyAdmin

change .env to add your local settings for web folders and ports to expose the services and then run:

docker-compose up -d build
